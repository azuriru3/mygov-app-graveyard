How we calculate this

Data comes from Malaysia's official GAMMA app registry, refreshed weekly. An app is "active-fresh" if updated within the last year, "active-stale" if 1-3 years, and "active-abandoned" if 3+ years — or if no update date was ever recorded. For apps on the App Store, we cross-check the registry's self-reported update date against Apple's own record via the public iTunes Search API, since the registry's own data isn't always accurate. Hall of Shame entries are abandoned apps we've hand-tagged as health, disaster-response, or food-safety related — not every old app is shamed here, only ones where staleness carries real risk.

This is independent analysis of public data, not an official government statement. Think an app is mischaracterized here? Open a correction.
